# mise

All dev tools managed by [mise](https://mise.jdx.dev/). Available both on host and inside [Distrobox](../distrobox/README.md) (shared home).

## Tools

**Languages** — go, rust, python 3.12, node (lts), bun, deno, java (zulu-21), gradle

**CLI** — bat, btop, eza, fd, fzf, hexyl, oha, ripgrep, croc, rclone, sing-box, tmux, upx, uv, yt-dlp, lazygit, protobuf

**Infra** — aws, gcloud, kubectl, helm, vercel, northflank

**Package managers** — yarn, pnpm

**Meta** — chezmoi, gh, git-lfs, neovim, starship, deepagents-code
