# mise

All dev tools managed by [mise](https://mise.jdx.dev/). Available both on host and inside [Distrobox](../distrobox/README.md) (shared home).

## Tools

**Languages** — go, rust, python 3.12, node (lts), bun, deno, java (zulu-21)

**CLI** — bat, btop, eza, fd, fzf, hexyl, oha, ripgrep, croc, rclone, tmux, upx, uv, yt-dlp, lazygit

**Infra** — aws, gcloud, kubectl, helm

**Meta** — chezmoi, git-lfs, neovim, starship
